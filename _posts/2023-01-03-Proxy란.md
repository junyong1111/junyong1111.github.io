**Proxy Sever의 기능**

1. **유저가 보낸 IP를 임의로 바꿀 수 있음**
2. **방화벽 기능**
3. **웹 필터 기능**
4. **캐쉬 데이터 제공**

**Proxy Sever의 사용 이유**

1. 회사나 직원들 집안에서 아이들 인터넷 사용 제어
    1. 특정 사이트 제한
2. 캐쉬를 이용하여 더 빠른 인터넷 이용 제공
    1. 미리 캐쉬에 자료를 보관하여 빠르게 처리가능
3. 더 나은 보안 제공
    1. 아이피를 가릴 수 있음
4. 이용 제한된 사이트 접근 가능
